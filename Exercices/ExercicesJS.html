1. Console log

Ajoutez une balise script dans votre fichier html

Dans cette balise, écrivez votre premier code console.log('hello world')

Vous devriez voir le message "hello world" affiché dans la console de votre inspecteur

2. Boutons et événements

Dans votre html, créer un bouton

Dans votre script, créer une fonction affichant du texte dans la console

Lier votre fonction à l'événement "click" de votre bouton

3. Listes de classes

Dans votre html, créer un bouton ainsi qu'un titre (h1)

Donner un id (unique !) à votre h1

Dans une feuille de style, créer une règle .primary

Cette règle va impacter le background-color et le color des éléments disposant de cette classe

Dans votre script, créer une fonction

Cette fonction doit récupérer l'élément h1 via son id et en modifier la liste de classes pour lui ajouter la classe primary

Lier votre fonction à l'événement "click" de votre bouton

Au click, votre titre doit changer de couleur de fond et de texte pour correspondre au style de la classe primary

4. Modification de texte

Dans votre html, créez une div et un bouton

Créer un script ajoutant du texte dans votre div

5. Addition

Dans votre html, créer deux input, une div "résultat" et un bouton

Créer un script récupérant le contenu des deux inputs, l'additionnant et l'injectant dans la div résultat

Essayer d'encoder deux chiffres dans les inputs et d'appuyer sur le bouton. Quel est le résultat ?

Essayer d'encoder "hello" dans le premier input et "world" dans le second. Appuyez sur le bouton. Quel est le résultat ?

Faites part de vos observations à votre professeur et/ou vos voisins de table

6. Modification de la dom

Dans votre html, créez un bouton

Créer un script ajoutant une div avec le texte 'hello world' se déclenchant au click sur le bouton

7. Modification de la dom : 2

Créer un script ajoutant une div au click sur un bouton

Cette div doit contenir un texte renseigné par l'utilisateur dans un input

7. Modification de la dom : 3

Créer un script ajoutant un nombre "n" de div au click sur un bouton

Ce nombre doit être encodé par l'utilisateur dans un input

Gérer les cas suivants

  • L'utilisateur encode un chiffre
  • L'utilisateur n'encode rien
  • L'utilisateur encode du texte
